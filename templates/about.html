🎬 About the Project

👨‍💻 Created by: Aman Pareek & Dinesh Suthar

🎓 University: Maharaja Ganga Singh University

💻 Department: Computer Science

📚 Course: MSc Computer Science (Lateral Entry)

This project is a Movie Recommender System 🎥 developed as part of the MSc Computer Science program. The aim of this system is to recommend movies based on movie title, genre, or actor 🎭. It provides top-rated movies ⭐ with detailed information including cast 👥, genres 🎭, ratings ⭐, and storylines 📝.

By combining data analysis 📊 and intelligent filtering 🤖, the system ensures that users receive relevant and engaging movie suggestions 🎯 that match their preferences.
